# Cards

## Combining stats
### Agility-Based Investigation (Foot)
These are primarily found in the Rogue and Survivor classes, allowing swift, sneaky characters to clear clues easily.

#### Stat-Adders (Intellect + Agility)
* [Lockpicks](https://arkhamdb.com/card/60305) (Level 0 / Level 1 Asset): The gold standard for Rogues. It allows you to investigate and add your Agility value to your Intellect value for the test.
* [Breaking and Entering](https://arkhamdb.com/card/07114) (Level 0 / Level 2 Event): Investigate by adding your Agility to your Intellect. If you succeed, you automatically evade an enemy at your location.
#### Stat-Replacers (Agility instead of Intellect)
* [Thieves' Kit](https://arkhamdb.com/card/09064) (Level 0 / Level 3 Asset): This tool lets you investigate using your Agility instead of Intellect. If you succeed, you also pocket a resource.  
* [Pilfer](https://arkhamdb.com/card/60315) (Level 0 / Level 3 Event): A high-impact event that lets you investigate using Agility. If you succeed, you discover 3 clues at once.

### Willpower-Based Investigation (Brain / Mind)
These are primarily found in the Mystic class (spells) and certain specialized Seeker/Neutral cards.  

#### Stat-Adders (Intellect + Willpower)
* [Read the Signs](https://arkhamdb.com/card/06117) (Level 0 Event): An incredible Mystic event. You investigate and add your Willpower value to your Intellect value. It also discovers 2 clues and ignores annoying location penalties or hidden negative text on chaos tokens.
* [Map the Area](https://arkhamdb.com/card/09048) (Level 0 Seeker Event/Insight): It attaches to a location. When anyone investigates that location, they can add their Willpower OR Agility to the test.
    
#### Stat-Replacers (Willpower instead of Intellect)
Mystics excel at turning clue-gathering into a test of mental fortitude using their Arcane slots:

* [Sixth Sense](https://arkhamdb.com/card/05158) (Level 0 / Level 4 Asset): Investigate using Willpower instead of Intellect. The upgraded version lets you investigate a location up to 2 connections away.
* [Clairvoyance](https://arkhamdb.com/card/60408) (Level 0 / Level 3 / Level 5 Asset): Investigate using Willpower to find 2 clues at once (though drawing a negative token can deal you horror).
* [Rite of Seeking](https://arkhamdb.com/card/02028) (Level 0 / Level 2 / Level 4 Asset): Investigate using Willpower to find multiple clues, but drawing a special token ends your turn immediately.
    
**Skill Card Tip:** Don't forget basic skill cards like [Curiosity](https://arkhamdb.com/card/05026) (Seeker) or [Rise to the Occasion](https://arkhamdb.com/card/02192) (Survivor), which don't change the underlying stat tested but allow you to commit your icons generously to pass standard Intellect tests using a heavy hand of cards!
